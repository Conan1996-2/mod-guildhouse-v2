This Addon will use a guild area in the world. Default predwefined areas are GM Island and a map not in use broken down into 3 areas.  
  
Chat Commands:  
  
.gh or .guildhouse  
    [asset]  
      [list]  
      [move]  
      [place] 
      [sell]  
      [store]  
    [house]  
      [sell]  
      [tele]/[teleport]  
    [npc]  
      [broker]  
      [salesman]  
    [shop]  
      [buy]  
      [catagories]  
      [list]
